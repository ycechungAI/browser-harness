# Make a video

Create a video only when the user asks or naturally nudges toward seeing the
work. Significant browser work alone is not consent.

Use the canonical workflow in
[`skills/browser-harness-video/SKILL.md`](../skills/browser-harness-video/SKILL.md).
It defines recording provenance, the supported edit-brief schema, privacy
rules, one-command review, and verified MP4 export. Do not hand-author
`composition.js` or fork the shared renderer.

If the video was requested before browser work, start an explicit recording
before the first action and retain its exact directory. Never substitute
`recordings --latest`, reenact a completed task, or invent missing evidence.
